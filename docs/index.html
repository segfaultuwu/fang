Fang is a minimalist CLI toolkit for network scanning, enumeration, and interactive security workflows. Inspired by Metasploit-style REPLs, built for fast terminal usage.

Features

Network Scanner

Discover hosts, scan ports, detect services and map local networks from the terminal.

Interactive REPL

Metasploit-style command interface with modules, options, sessions and reusable workflows.

Modules

Load scanner, enum and utility modules without bloating the core CLI.

Minimal Design

No heavy UI. No useless noise. Just fast terminal-first tooling.

Install

cargo install fang-toolkit
